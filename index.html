BibleTag API

What is the BibleTag API

This service supports a REST API for collaboratively tagging the Bible.

Make your first API calls

GET VERSES BY TAG

cURL Example:

curl http://bibletag.xyz:8080/tag/love

POST A TAG

cURL Example:

          curl -X POST \
            -H "Content-Type: application/json" \
            -H "Cache-Control: no-cache" \
            -H "Postman-Token: aced34fd-2249-bff2-af6a-905ad2eeddfa" \
            -d '{
                "tag": "worry",
                "book": "1 peter",
                "chapter": 5,
                "startVerse": 7,
                "endVerse": 7
              }' "http://bibletag.xyz:8080/tag"
