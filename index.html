BibleTag API

What is the BibleTag API?

The BibleTag-API is a REST API that enables developers to retrieve Bible verses (in JSON format) based on crowdsourced tagging of of the Bible. The API also allows developers to tag Bible verses from within their apps and contribute to the repository of tags. For example, you or your users may not know a relevant Bible reference (e.g., John 3:16). However, you may know that your would like to display or serve a Bible verse or verses related to "love," "worry," or "encouragement." The BibleTag-API supports this type of interaction.

POST A TAG

cURL Example:

          curl -X POST \
            -H "Content-Type: application/json" \
            -H "Cache-Control: no-cache" \
            -H "Postman-Token: aced34fd-2249-bff2-af6a-905ad2eeddfa" \
            -d '{
                "tag": "worry",
                "book": "1 peter",
                "chapter": 5,
                "startVerse": 7,
                "endVerse": 7
              }' "http://bibletag.xyz:8080/tag"

Response:

          {
            "code": 200,
            "text": "Tagged Passage"
          }
